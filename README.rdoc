== INFORMACIÓN

Aplciación web la cual permite encontrar todas las ofertas de trabajo relacionadas con Ruby on Rails. 

La información de las ofertas de trabajo es obtenida a través de la tecnica scraping en diferentes plataformas web de ofertas de trabajo.
